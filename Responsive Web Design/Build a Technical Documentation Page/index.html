Introduction

JavaScript is a programming language used for web development, server-side scripting, and more.

It enables interactive web pages and is an essential part of web applications.

JavaScript is a lightweight, interpreted language.

It is designed for creating network-centric applications.

  • Client-side JavaScript
  • Server-side JavaScript
Variables

Variables in JavaScript are used to store data values.

var x = 5; let y = "hello";

You can declare variables using var, let, and const.

  • Global variables
  • Local variables
Functions

Functions are one of the fundamental building blocks in JavaScript.

A function is a block of code designed to perform a particular task.

function myFunction() { return "Hello World"; }

Functions are defined using the function keyword.

  • Function declaration
  • Function expression
Loops

Loops allow you to repeat code until a certain condition is met.

JavaScript supports several loop types including for, while, and do-while.

for (let i = 0; i < 5; i++) { console.log(i); }
  • For loop
  • While loop
  • Do-while loop
Arrays

Arrays are used to store multiple values in a single variable.

let fruits = ["Apple", "Banana", "Mango"];

Arrays in JavaScript are flexible and dynamic.

  • Modifying arrays
  • Iterating over arrays
